Q 1：Python有哪些特点和优点？

解释型语言、面向对象、简单易于阅读、开源、社区强大、动态类型语言、胶水语言。

Q 2：深拷贝和浅拷贝之间的区别是什么？

深拷贝是单独开辟一块内存空间存放变量，和原来的变量互不影响；浅拷贝是复制内存地址，和原来的变量互相影响。

Q 3. 列表和元组之间的区别是？

元组是不可变类型，列表是可变类型。

Q 4. 解释一下Python中的三元运算子

(True执行的代码) if True else (False执行的代码)。

Q 5. 在Python中如何实现多线程？

一、threading.Thread().start()
二、class(threading.Thread): def run(self): ...
由于python中存在GIL锁，所有线程都是在cpu上轮流运行，由于切换速度快，所以看上去像并行运行的。

Q 6. 解释一下Python中的继承

被继承的类称为父类，继承的类称为子类，子类会获取父类所有的属性和方法，可以有单继承，多继承等。

Q 7. 什么是Flask？

一个轻量级的web应用框架，其WSGI服务器是Werkzeug，模板引擎是jinjia2。

Q 8. 在Python中是如何管理内存的？

栈内存：存放变量和内存地址，堆内存：存放值。
主要采用引用计数器进行计数，当引用计数为0的时候，就会回收内存，del语句可以删除引用计数，局部变量超出局部部分也会删除引用计数。

Q 9. 解释Python中的help()和dir()函数

help(模块)可以查看模块的使用帮助；
dir(模块)可以查看模块的属性和方法列表。

Q 10. 当退出Python时，是否释放全部内存？

不会，退出python时，垃圾回收机制也会退出，所以没法释放全部内存，这些没有释放的内存是由操作系统负责释放的。

Q 11. 什么是猴子补丁？

就是在模块加载后，通过对模块中部分方法的重新赋值，动态修改方法功能。

Q 12. Python中的字典是什么？

是一个拥有键值对的存储类型，属于可变数据类型，可以用推导式创建。

Q 13. 请解释使用*args和**kwargs的含义

args是不定长参数元组，用于接收位置参数；kwargs是不定长参数字典，用于接收关键字参数。

Q 14. 请写一个Python逻辑，计算一个文件中的大写字母数量

print(len([w for w in open(...).read() if w.isupper()]))
print(len(list(filter(lambda w: w.isupper(), open(...).read()))))

Q 15. 什么是负索引？

从右到左从-1，-2开始的索引，可以用于切片。

Q 16. 如何以就地操作方式打乱一个列表的元素？

print(random.shuffle(list)）

Q 17. 解释Python中的join()和split()函数

join()函数可以将一个对象每个元素中间插入字符串，split()函数可以以指定字符串切割字符串。

Q 18. Python区分大小写吗？

区分的。

Q 19. Python中的标识符长度能有多长？

可以是任意长度。

Q 20. 怎么移除一个字符串中的前导空格？

print(str.lstrip())

Q 21. 怎样将字符串转换为小写？

print(str.lower())

Q 22. Python中的pass语句是什么？

占位符，是一个不会被执行的语句，省略号（...）也可以用来占位。

Q 23. Python中的闭包是什么？

在一个函数内定义一个嵌套函数，并返回这个函数，这个嵌套函数内又会访问其外部函数的变量，这样操作可以使得程序更加简单易读。

Q 24. 解释一下Python中的//，%和 ** 运算符

//是向下整除取整；%是除法取余；**是乘方运算。

Q 25. 在Python中有多少种运算符？解释一下算数运算符。

共有七种运算符。
算数运算符（+-*/等）、关系运算符（><==等）、赋值运算符（=等）、逻辑运算符（and、or、not）、位运算符（&|^~<<>>）、成员运算符（[not ]in）、身份运算符（is[ not]）。
算术运算符包括加减乘除、乘方、向下整除取整、除法取余。

Q 26. 解释一下Python中的关系运算符

关系运算符包括大于>、小于<、等于==、不小于>=、不大于<=、不等于!=。

Q 27. 解释一下Python中的赋值运算符

基础的赋值运算符=，可以在前面加上算数运算符快速运算。

Q 28. 解释一下Python中的逻辑运算符

返回真的情况：真 and 真、假 or 真、not 假

Q 29. 解释一下Python中的成员运算符

判断一个值是否在另外一个值里。

Q 30. 解释一下Python中的身份运算符

判断一个值是否是和另外一个值相同，或者一个对象是否是另外一个对象的子类。

Q 31. 讲讲Python中的位运算符

位与&、位或|、位异或^、位取反~、左移位<<、右移位>>等运算符。

Q 32. 在Python中如何使用多进制数字？

0b或0B代表二进制，0o或0O代表八进制，0x或0X代表十六进制。

Q 33. 怎样获取字典中所有键的列表？

list(dict.keys())或者直接llist(dict)。

Q 34. 为何不建议以下划线作为标识符的开头

因为这是python约定私有变量的命名方式。

Q 35. 怎样声明多个变量并赋值？

a, b, c = 1, 2, 3或者(a, b, c) = (1, 2, 3)；
如果赋值的是相同的值则：a=b=c=1；
多变量赋予同一值，实际上是浅拷贝了一份内存地址，如果是a=b=[1,2]执行b.append(3)则a=[1,2,3]。

Q 36. 元组的解封装是什么？

a, b, c = (1, 2, 3)将对应位置的值赋值给对应位置的变量就是解封装；
d = 1, 2, 3将多个值赋值给一个变量就是封装（变成元组）。
